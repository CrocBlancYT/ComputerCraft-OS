# ComputerCraft-OS
A simplistic CC Operating System made for giving an interface and tools for developpers.

## What it offers :

- better control flow :
task & signals libraries

- built-in programs and librairies for simple usage :
inputs, screen methods, vector extension, bitmap, ... & third-party librairies :
(hashing functions, png, deflate, ...)

- admin login & file explorer
